Corrigir/Mesclar Arquivos HTML

Use esta ferramenta para unir um arquivo original e um arquivo com correções. Ideal para corrigir falhas em um arquivo HTML gerado pela conversão de PDF (páginas faltando ou com conteúdo incompleto).

Como usar:

  1. Volte para a aba Converter PDF.
  2. Envie o mesmo PDF novamente, mas especifique apenas as páginas com erro (ex: 4, 9-11).
  3. A conversão irá gerar um arquivo HTML menor (apenas as páginas reprocessadas).
  4. Use o formulário abaixo:
    • HTML Original: O arquivo grande gerado na primeira conversão.
    • HTML com Correções: O arquivo pequeno recém-gerado.
